# n8n migration exports

Экспорты воркфлоу n8n (документация / резерв).

⚠️ **Перед импортом в n8n**: плейсхолдер `__SUPABASE_ANON_KEY__` в полях
`apikey` / `Authorization` нужно заменить на реальный Supabase **anon**-ключ
(или, правильнее, завести n8n-credential и ссылаться на него). Сырой ключ
намеренно вычищен — секреты не хранятся в git.

- `ai-agent.json` — КотЭ AI-агент с памятью
- `schedulers.json` — напоминания о туре / запрос отзыва
