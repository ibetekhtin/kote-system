## 2. ГЛАВНЫЙ ФАЙЛ БОТА

**Путь:** `bot/index.js` (260 строк, Node.js)

**Технология:** Telegraf 4.16 + session middleware

**Архитектура:**
```
User → Telegram → bot/index.js → bot/ai.js (Gemini) → Supabase → ответ
                  ↑ middleware: загружает клиента из Supabase в ctx.state
```

**Команды:**
- `/start` — выбор направления (market:phuket, market:pattaya, market:bali, market:dubai)
- `/services` — список услуг по типу (tour/transfer/rental)
- `/bookings` — мои бронирования
- `/help` — справка
- Любой текст → fallback в AI (КотЭ через Gemini)

**Callback handlers:**
- `market:(id)` — выбор рынка → upsertClient
- `type:(tour|transfer|rental)` — список услуг по типу
- `book:(serviceId)` — создание брони через Supabase
- `cancel_booking:(id)` — отмена брони

## 3. AI МОДУЛЬ

**Путь:** `bot/ai.js`

**AI Engine:** Google Gemini 2.0 Flash (НЕ Claude)

**Как работает:**
1. Системный промпт загружается из `ai/kote_prompt.txt`
2. Берутся последние 10 сообщений из Supabase
3. Формируется prompt = systemPrompt + история + "КотЭ:"
4. Отправляется в Gemini → текстовый ответ

**Важно:** НЕТ инструментов (tools), НЕТ function calling. Просто текстовый генератор.

## 4. СИСТЕМНЫЙ ПРОМПТ КОТЭ

**Путь:** `ai/kote_prompt.txt`

**Содержимое (полностью):**
```
Ты — КотЭ, AI-помощник проекта «Нестандартный Отдых».

## Личность
- Дружелюбный, эмпатичный, короткие ответы
- Общаешься на русском языке
- Emoji 🐾 в начале ответов

## Принципы
1. Сначала польза. Потом продажа.
2. Никогда не выдумывай данные — работай только через Supabase.
3. Превращай запрос в бронирование — помогай человеку забронировать.
4. Краткие ответы — не пиши больше 3 предложений.

## Доступные рынки
- phuket — Пхукет, Таиланд
- pattaya — Паттайя, Таиланд
- bali — Бали, Индонезия
- dubai — Дубай, ОАЭ

## Правила
- Если спрашивают про услуги — ищи через getServicesByMarket
- Если хотят забронировать — помогай создать booking
- Если вопрос экстренный (SOS) — давай номера: полиция 191, скорая 1669
- Не выдумывай цены, описания, availability — только из Supabase
- Не выдавай информацию из других рынков если клиент выбрал свой

## Формат ответа
- Начинай с 🐾
- Отвечай на русском
- Максимум 3 предложения
- В конце — предложение действий: «Хочешь забронировать?»
```

## 6. КАК ЗАПУСКАЕТСЯ

```bash

docker compose up -d   # 3 контейнера: kote-bot, kote-backend, n8n
```

**Docker Compose (3 сервиса):**
1. `kote-bot` — Node.js Telegram бот
