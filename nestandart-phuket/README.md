Brutalist editorial + глитч-эффекты + параллакс. Чёрный бриллиант фон, кислотно-салатовый + кислотно-оранжевый акценты.

## 🚀 Запуск (2 минуты)

### 1. Замени контакты бота

Открой `js/app.js` → строка 4 → замени `'your_bot'` на имя своего Telegram-бота.

Телефон **+66 80 489 4595** уже стоит везде. Если поменяется — найди по поиску `66804894595` и замени.

### 2. Залей на Netlify

**Способ A — обновить твой существующий сайт:**
1. Зайди в Netlify → твой сайт (`yo-yo-yo-yo-yo-yo.netlify.app`) → Deploys
2. Перетащи папку `nestandart` на страницу деплоев
3. Через минуту твой старый сайт заменится новым — ссылка останется та же

**Способ B — новый деплой:**
1. https://app.netlify.com/drop → перетащи папку
2. Получишь новую ссылку

## 🔍 SEO — что делать после запуска

### Шаг 1: Подключи Google Search Console
1. https://search.google.com/search-console → Добавить ресурс
2. Подтверди владение (через DNS или HTML-файл)
3. Отправь sitemap.xml в разделе "Файлы Sitemap"

### Шаг 2: Подключи Яндекс.Вебмастер
1. https://webmaster.yandex.ru → Добавить сайт
2. Подтверди владение
3. Отправь sitemap.xml

### Шаг 3: Замени URL в файлах
В sitemap.xml, robots.txt и всех HTML — замени `nestandart-otdyh.netlify.app` на свой реальный домен.
